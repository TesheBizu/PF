# Teshome Bizuayehu — Full Stack Portfolio

A personal portfolio and admin dashboard built with the MERN stack. Features a public-facing portfolio site and a password-protected admin panel for managing projects, skills, experience, and messages.

- **Portfolio** — [teshome.vercel.app](https://teshome.vercel.app)
- **Admin** — [pf-admin-port.vercel.app](https://pf-admin-port.vercel.app) (restricted)
- **API** — [pf-o6tc.onrender.com](https://pf-o6tc.onrender.com)

## Tech Stack

**React**, **Redux Toolkit**, **Vite**, **Node.js**, **Express**, **MongoDB Atlas**, **Socket.io**, **JWT**, **Cloudinary**, **Vercel**, **Render**
